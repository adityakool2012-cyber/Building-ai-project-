# Building-ai-project-

## Summary
This project predicts whether an animal is a dog or cat based on its weight. If weight > 10 kg, it predicts dog, else it predicts cat. This is my first AI project for the Building AI course.

## Background
I made this simple rule-based predictor to understand basic if-else logic in AI. It uses weight as the only feature to classify between dog and cat.

## How is it used?
Run the Predictor.py file and input the weight of the animal. The program will tell you if it's likely a dog or cat based on the 10kg rule.

## Data sources and AI methods
No external data used. Simple if-else rule: weight > 10 → Dog, else → Cat.
